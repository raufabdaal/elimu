# P7 Social Studies Topic Map
## Topic: Maps and Location

**Curriculum Position**: This is typically **Topic 1** in P7 Social Studies.

### Sub-Topic Breakdown

| Sub-Topic | Description | Priority | Suggested Questions |
|-----------|-------------|----------|---------------------|
| **1. Elements of a Map** | Title, key/legend, scale, compass, grid references | High | 5–6 |
| **2. Map Reading Skills** | Interpreting symbols, directions, distances | High | 5–6 |
| **3. Grid References** | 4-figure and 6-figure grid references | High | 4–5 |
| **4. Scale** | Understanding and using different scales | Medium | 3–4 |
| **5. Compass Directions** | Cardinal and intercardinal points | Medium | 3–4 |
| **6. Location of Places** | Finding places using maps, latitude & longitude basics | Medium | 4–5 |
| **7. Types of Maps** | Physical, political, thematic maps | Low | 2–3 |

**Target Total**: 26–30 questions (can be split into 2–3 flexible sessions)

---

### Recommended Session Structure (Flexible)

- **Session 1**: Elements of a Map + Compass Directions (10–12 questions)
- **Session 2**: Grid References + Scale (10–12 questions)
- **Session 3**: Map Reading Skills + Location of Places + Types of Maps (8–10 questions)

---

### Quality Standards
- Heavy use of real map interpretation style questions
- Include simple grid reference practice
- Focus on Uganda-relevant map examples where possible
- Educational explanations that teach map skills
